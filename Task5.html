Task 5

Plane mirror imaging: reflect pixel coordinates to plot a virtual image

Overview

Import an image (object) positioned to the right of a vertical mirror at x = 0, compute its virtual image by reflecting pixel coordinates across the mirror, and display both object and virtual image. The app also draws a physically correct reflection point and viewing ray to illustrate the law of reflection.

Key Theory

A plane mirror produces a virtual image the same distance behind the mirror as the object is in front. For a vertical mirror at x = 0, each object point (x, y) maps to the virtual image point (−x, y); the image is laterally inverted but upright and the same size. This pixel-wise reflection is what the program uses to render the virtual image.

Rays obey the law of reflection: angle of incidence equals angle of reflection, measured from the normal to the surface. A convenient way to find the correct strike point S on the mirror for a viewer at B is the mirror‑image method: reflect the eye B to B′ across the mirror and draw the straight line A→B′; where it meets the mirror is S. This is equivalent to enforcing equal angles at S and is used here to draw the incident/reflected rays and confirm θ₁ = θ₂ numerically.
