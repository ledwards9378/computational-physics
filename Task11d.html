Task 11d

Primary and secondary rainbow arcs at sea level for different Sun elevations

Overview

Draw the rainbow arcs as seen from sea level for several solar elevation angles α (5°, 20°, 30°, 40°), with an interactive panel to vary α. The arcs are coloured by wavelength using the same dispersion model as earlier tasks.

Key Theory

Directions of strongest scattering lie on a cone around the antisolar direction with half‑angle εmin(λ). From the observer’s eye, points on this cone above the horizon (z > 0) form visible arcs. We project the cone to a screen using a simple perspective (onto the plane x = 1) so the horizon is a straight line.

  • For each λ, compute εmin for primary and secondary using θ⋆(n(λ)).
  • Form the cone relative to the antisolar vector a at altitude −α; trace many azimuths.
  • Keep only points with positive altitude; project to the screen to plot the arcs.
