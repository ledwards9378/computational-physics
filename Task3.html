Task 3

Fermat’s principle: least-time path for reflection from a plane surface

Overview

Calculate the travel time for light to go from point A to point B by reflecting off a plane at y = 0 via a point S at horizontal position x, plot t(x), and identify the minimum. The x at the minimum corresponds to the physical path selected by Fermat’s principle.

Key Theory

Fermat’s principle states that the path taken by light between two points makes the optical travel time stationary (usually a minimum). In a uniform medium of refractive index n, the optical path length is OPL = n·s (n times the geometric distance s), so the time is t = OPL / c = (n/c)·s.

For reflection at a plane y = 0, consider A = (0, y) and B = (L, y). Constraining the ray to touch the plane at S = (x, 0), the total time as a function of x is t(x) = (n/c) [ √(x² + y²) + √((L − x)² + y²) ]. Minimising t(x) gives dt/dx = 0 ⇒ x/√(x² + y²) = (L − x)/√((L − x)² + y²). Writing sinθ₁ = x/√(x² + y²) and sinθ₂ = (L − x)/√((L − x)² + y²) shows sinθ₁ = sinθ₂, hence θ₁ = θ₂: the law of reflection (angle of incidence equals angle of reflection).

In the symmetric case used here (same height y for A and B), the minimum occurs at x = L/2. The t(x) curve is smooth and convex, with a single minimum; marking this on the plot provides a clear numerical confirmation of Fermat’s principle.
