Task 1b

Model the refractive index of liquid water across the visible band using an empirical n–f relation (405–790 THz)

Overview

Compute the refractive index of water as a function of optical frequency f over 405–790 THz (equivalently wavelength λ ≈ 740–380 nm) using the given empirical relation, and produce a plot of n against f (or λ). Optionally colour the curve by the visible spectrum bands.

Key Theory

Refractive index n determines how much light slows down in a medium: v = c / n. In transparent materials like water, n varies with frequency (dispersion). For this task we use an empirical fit that is accurate in the visible range and convenient to compute from frequency.

Frequency–wavelength conversion in vacuum follows f = c / λ, where c ≈ 2.99792458 × 10⁸ m·s⁻¹. Useful conversions: λ[m] = c / f[Hz]; λ[nm] = 299792.458 / f[THz]. The visible band 405–790 THz corresponds to λ ≈ 740–380 nm.

Empirical model for water (visible band): (n² − 1)^(−2) = 1.731 − 0.261 · (f / 10¹⁵ Hz)². If you work with f in THz, use (f/10¹⁵ Hz) = f[THz]/1000. To recover n, compute x = 1.731 − 0.261 · (f/1000)², then use n = sqrt(1 + x^(−1/2)). Numerically guard against tiny negative x at the extremes by clamping x to a small positive value before taking powers.

Expected behaviour across the visible: n increases slightly with frequency (towards violet). Typical values are near 1.331 at the red end (≈ 405 THz, λ ≈ 740 nm) and up to ≈ 1.339 near the violet end (≈ 790 THz, λ ≈ 380 nm). A good check is around the sodium D line (589 nm, ≈ 508 THz) where n ≈ 1.333 for liquid water at room temperature.

Optional visualisation: colour the plotted points/segments by approximate visible colour bands in THz, for example 405–480 (red), 480–510 (orange), 510–530 (yellow), 530–600 (green), 600–620 (cyan), 620–680 (blue), 680–790 (violet). Interpolating between band colours produces a smoother gradient.
